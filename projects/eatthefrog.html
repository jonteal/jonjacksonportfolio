Eat the Frog

This was my first collaborative experience in a project. The work was divided among our group into sections where each felt strongest at the time. I focused on the weather API and front-end functionality.

The application is a task manager that is designed to give the user a point-based reward system for every task they accomplish, as well as an interactive landing page that includes a personalized weather component and joke generator for amusement purposes.
