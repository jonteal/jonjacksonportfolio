Random Password Generator

This project was built using the basic principles of Javascript. It has a minimal front-end HTML and CSS design, and is a simple exploration of how to use concepts like Math.random in order to achieve randomization in its outcomes. As it was one of my first projects using Javascript, it holds a special place in my heart and I often come back to it to look at the code in hopes I can learn something new or spot an opportunity for refactoring.

The tool is a very easy-to-use application for a user to create a completely random password that can be anywhere from 8 - 128 characters including uppercase letters, lowercase letters, numbers, or special characters. The user can opt out of using any of those options and once they complete the prompts, they will have a completely random password everytime. I love simple yet powerful concepts like the ones used in this project!
