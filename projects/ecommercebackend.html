E-Commerce Back-End

This project was built using the NPM depenencies Dotenv, Express, MySql2, and Sequelize in order to configure an Express.js API to use Sequelize to interact with a MySQL database using Object Relational Mapping. It was a blast bringing all these tools together to create a working back end for an e-commerce website, and also considering how many ways these same knowledge and methodologies could be applied.

The purpose of this application was to build an back end for supporting and managing categories, products, and tags for an e-commerce website. Through this back end, a user could easiy create GET, POST, PUT, and DELETE requests in order to execute data retrieval, posting, updating, and deleting function.
